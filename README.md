# Satellite Control Center

AI-assisted satellite telemetry dashboard for an AI Data & IoT DevOps portfolio project.

## Tech Stack

- Go IoT simulator for generating satellite telemetry.
- SvelteKit dashboard for the web interface and server-side API routes.
- Gemini API for LLM-based satellite operations diagnosis.
- Local JSON telemetry file generated by the simulator.
- Python RAG engine remains available for a later LangChain/RAG phase.

## Run the IoT Simulator

```powershell
cd iot-simulator
go run .
```

The simulator writes runtime telemetry to `logs/telemetri.json`.

## Run the Dashboard

```powershell
cd dashboard
npm install
npm run dev
```

Open the SvelteKit URL shown in the terminal, usually `http://localhost:5173`.

## Configure Gemini

1. Open `/settings` in the dashboard.
2. Paste your Gemini API key.
3. Keep the default model `gemini-2.5-flash` or choose another Gemini model.
4. Save settings.

The API key is stored server-side in `dashboard/.runtime/settings.json`, which is ignored by Git.

## API Routes

- `GET /api/telemetry/latest`
- `GET /api/telemetry/history`
- `GET /api/settings/status`
- `POST /api/settings`
- `POST /api/diagnosis`
